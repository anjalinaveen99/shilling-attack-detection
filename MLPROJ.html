Key Metrics Explained

GIAD (Group Item Attention Degree)

= (Users in group rating item) / (Total users rating item)

Higher value = More suspicious

UA (User Activity)

= (Synchronous ratings) / (Total ratings per user)

Higher value = Coordinated behavior

Suspicious Degree (SD)

= (GIAD + UA) / 2

Combined metric for attack detection

Detection Threshold

= mean(SD) + std(SD)

Separates attacks from normal behavior

  • ‚úì Density-based clustering algorithm - no predefined cluster count needed
  • Parameters:

  • Detection Process:

    • Compares with threshold
    • Groups above threshold = Attack Groups
